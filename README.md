# YunNote 作者：王旭

## 这是一个基于 Android 和 MySql 的云笔记本程序。


### 实现功能:   
  1. **用户的注册登录退出模块**操作
  2. **用户信息的查找更改**操作
  3. **笔记列表的查看**操作
  4. **笔记列表的删改**操作
### 本软件的编写环境：
	Android Studio 3.2.1
	Gradle Version：4.6
	Android Pulgin Version：3.2.1
	Java Version : 1.8.0_152
	Android Sdk Version：API 28: Android 9.0 (Pie)
